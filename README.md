## Hangperson [Live Link](http://hangperson.com)
Realtime online two-player Hangman with chat functionality.
Built with a Golang backend and a React/Redux frontend.

## Stack
- Frontend: React/Redux
- Backend: Golang
- Database: PostgreSQL
